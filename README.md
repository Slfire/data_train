Data Engineer Training

## 🎯 Objectif

Reconversion vers le métier de **Data Engineer**, via un parcours 100% gratuit et autodidacte, structuré en phases progressives (Linux/Git → SQL → Python → Modélisation → ELT/dbt → Orchestration/Airflow → Traitement distribué/Spark → Qualité & CI/CD → Monitoring).

Ce dépôt centralise l'ensemble des exercices, projets et livrables réalisés au fil du parcours.
